## Color Coding Guide

```
Primary Amount (Original Currency)
  Text: Large, Bold
  Color: Asset-specific (Green for cash, Orange for crypto, Purple for stocks)

Secondary Amount (Converted)
  Text: Smaller, Regular weight
  Color: Gray (text-gray-500)
  Prefix: ≈ (approximately symbol)

Examples:
  $45,234           ← Primary (original)
  ≈€41.5K           ≈ Secondary (converted)

  £12,500           ← Primary (original)
  ≈$15.8K           ← Secondary (converted)
```

## Compact vs Full Format

```
Large Values - Use Compact:
  $1,234,567  →  ≈€1.1M
  $45,678     →  ≈€41.8K
  $987        →  ≈€905

Small Values - Full Format:
  $12.50      →  ≈€11.48
  $1.23       →  ≈€1.13
```
